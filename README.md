# DeepSeek_Survey--R--Code
R analysis script for 2026 DeepSeek AI perception questionnaire survey in Panzhihua; includes data cleaning, reliability analysis, factor analysis and visualization codes.

## Supporting Raw Dataset
All anonymized survey responses, statistical tables, research figures and supplementary documentation used in this project are archived open-access on Zenodo:
- Fixed Version 1.0 Dataset DOI: https://doi.org/10.5281/zenodo.21485667
- Persistent All-Version Global DOI: https://doi.org/10.5281/zenodo.21485666
- Full Zenodo Record Page: https://zenodo.org/record/21485667
- Dataset License: Creative Commons Attribution 4.0 International (CC BY 4.0)

## Project Overview
This repository contains complete reproducible R code for the cross-sectional social survey studying public attitudes toward DeepSeek generative AI in Panzhihua, China. All personal identifiable information has been removed from the matched Zenodo dataset to protect participant privacy.
